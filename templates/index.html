Sensor de Temperatura

Temperatura en tiempo real 
El valor numérico muestra la temperatura actual del sensor (se actualiza cada segundo) y se vuelve de color rojo cuando se encuentra en estado de cocción. En la gráfica aparecen los ultimos 60 puntos (~30 minutos). Cada punto se guarda en intervalos de 30 segundos, excepto al activar y desactivar la cocción, que se guardan de forma inmediata.

Temperatura día 
En esta gráfica se muestran las temperaturas del día seleccionado. Para seleccionar un dia simplemente hay que pulsar la fecha situada debajo de este titulo. El boton de la derecha permite recargar la misma fecha (por ejemplo, si es el día actual y hay nuevos datos). Esta gráfica no se actualiza automaticamente.

Aquí se puede descargar un fichero de tipo ".csv" con todos los datos del mes seleccionado. Este fichero se puede abrir directamente con Excel o con el Bloc de notas. Se añaden nuevos datos al mes actual cada 30 segundos.
